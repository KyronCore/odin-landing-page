# Odin Landing Page

## Project Overview
This project is part of The Odin Project curriculum. It is a static landing page built using HTML and CSS.

The goal is to recreate a given design using:
- HTML structure
- CSS styling
- Flexbox layout

No JavaScript is used in this project.

---

## Skills Demonstrated
- Writing semantic HTML
- Structuring a full web page
- Using CSS for layout and styling
- Flexbox for alignment and spacing
- Working with colors, spacing, and typography
- Building a page from a design reference
